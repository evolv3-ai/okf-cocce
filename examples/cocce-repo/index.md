# cocce-repo

This example shows a small OKF-COCCE pattern.

## Files

- `index.md` is the example entrypoint.
- Concept documents use YAML frontmatter with a non-empty `type` field.

A COCCE-enabled repository includes `okf/cocce.md`, categories, objects, capabilities, connections, events, policies, and playbooks.
